# z-nodejs-ms-core

Core layer for fast and productive building of microservices using TypeScript and express

## Getting Started

Modules

### Prerequisites

- NodeJS 14

### Dependencies

- neo4j-driver
- tslog
- express

### Installing

Install package

```bash

npm i @zurckz/nodejs-ms-core

```
